SolutionsDepot

SolutionsDepot is digital warehouse of solutions.

You can store your solution in SolutionsDepot.

You can search for solutions.

SolutionsDepot only excepts solutions in JSON FDC format.

We follow IEEE JSON solution standards described on https://www.ieee.org/index.html.

This box provides schema of a solution in JSON.

Following is the minimum list of object in schema needed to upload file.

  • "basics" ObjectType: {}
  • "work" ObjectType: [{}]
  • "skills" ObjectType: [{}]
  • "education" ObjectType: [{}]
