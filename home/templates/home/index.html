SolutionsDepot

SolutionsDepot is digital warehouse of Solutions.

You can upload and search for certified solutions in SolutionsDepot.

SolutionsDepot only excepts solutions in JSON format.

Please follow these steps for submitting a solution.

1) Size Solution: http://www.emc.com/auth/rcoll/tool/vxrail-4.5-advanced-sizing-tool-for-14g.xlsm.

https://www.emc.com/auth/rcoll/tool/vxrail-4-5-advanced-sizing-tool-for-13g.xlsm.

https://mainstayadvisor.com/go/emc/vxrail/.

2) Design & Export using Fabric Design Center https://fdc.emc.com.

3) Test & Certify solution.

4) Publish solution via upload.

or simply find & use previously tested solution using search.

REFERENCE VxRail Enablement Center:

https://www.emc.com/auth/rpage/ec-for-vce-vxrail.htm.
